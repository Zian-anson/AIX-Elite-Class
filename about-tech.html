01 采集 — 微信本地数据解密

wx-cli（@jackwener/wx-cli）

微信 macOS 本地数据查询 CLI，Rust 单二进制。通过扫描 WeChat 进程内存提取 SQLCipher 加密密钥，实时解密本地数据库。

流程：codesign（移除 Hardened Runtime）→ sudo wx init → 提取 18 个数据库密钥
⚠️ 需要 macOS ad-hoc 签名 + 完全磁盘访问权限

macOS 代码签名

sudo codesign --force --deep --sign - "/Applications/微信.app" — 移除 WeChat 的 Hardened Runtime 保护，使进程内存可被外部工具读取。

注意：重新签名后微信需要重启登录，数据不受影响

02 清洗 — Python 多重过滤管道

原始导出约 1,881 条消息，经过 5 道筛子过滤：

1️⃣ 系统消息（入群/撤回/管理员变更）
2️⃣ XML 元数据（CDATA、appmsg 等微信内部结构）
3️⃣ 纯表情/庆祝回复（[表情][庆祝][烟花]等）
4️⃣ 接龙去重（同一份签到链 20+ 人重复，只留第一份）
5️⃣ 空泛回复（"好的""收到""谢谢老师"）

最终 1,140 条有效消息，过滤率 ~39%
